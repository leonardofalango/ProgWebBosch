Projeto Processing

O projeto feito no Processing3, é um ambiente 3D que cria uma quantidade de cubos definida antes de iniciar o canvas, a cada canvas iniciado, as posições dos cubos, a cor deles e o tamanho são aleatórios. Você pode viajar por esse ambiente 3D por padrão devagar, mas com o click esquerdo, ou click direito, você tem a liberdade de escolher acelerar ou voltar. Além disso, os cubos possuem um tamanho mínimo e variam de tamanho conforme você fala no microfone.
